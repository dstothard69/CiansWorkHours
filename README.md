# CiansWorkHours
Logging Cians Work Hours
Monthly Hours Log - Offline Web App

How to use:
1. Open index.html in Chrome, Edge, Safari or Firefox.
2. Enter your employee details.
3. Log each working day using start/end times or manual hours.
4. Select the month you want to report on.
5. Use "Export spreadsheet CSV" to create a payroll spreadsheet.
6. Use "Create PDF / print" and choose "Save as PDF" in the print window.

Important:
- Your data is saved locally in your browser on your device.
- It does not upload anything online.
- Use Export backup JSON if you want a backup of your entries.
